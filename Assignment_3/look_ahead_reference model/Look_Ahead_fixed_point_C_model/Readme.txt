iir_filter.c ====> is the file containing the C code model of the filter fixed point architecture with the Look Ahead improvement

iir_filter.exe ====> is the executable version of it

samples.txt ====> is the file generated by Matlab containing the average of the two waveforms created

output_20_LK.txt ====> is the file containing the results of the filter C model in fixed point, with the Look Ahead improvement
